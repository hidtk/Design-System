Matte gradient panel with a hairline border — the default container for grouped content.

```jsx
<Card>
  <span className="akm-eyebrow">Телеком</span>
  <h3>Измерения до 67 ГГц</h3>
</Card>
```

Variants: `default`, `active` (blue border + soft glow — selected/highlighted), `alert` (rose pain palette with left edge stripe — use ONLY for risks, problems, negatives). Cards and pills replace bulleted lists.
