Pill-shaped action button (Montserrat 700) — use for any clickable action; `primary` for the single main action per view.

```jsx
<Button variant="primary" size="md" onClick={go}>Получить расчёт</Button>
```

Variants: `primary` (solid blue, dark text), `secondary` (hairline outline), `ghost` (blue text only). Sizes: `sm` / `md` / `lg`. Supports `icon` (leading thin-line glyph) and `disabled`. Never more than one primary button competing in a single scene.
