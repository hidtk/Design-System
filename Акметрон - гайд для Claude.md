## Кто такой Акметрон

Инженерно-техническая компания: контрольно-измерительное оборудование (test &
measurement) и радиоэлектроника. Бренд-знак — тёмно-синий (`#004180`) круг из четырёх
переплетённых пазлов. Тон визуала — приборный, светлый, функциональный.
**Цвет несёт смысл, а не украшает: синий = «смотри сюда», розовый = «опасность»,
серый = вторичное. Никогда не раскрашивай ради разнообразия.**

Семь направлений каталога: Телеком · High Speed Digital · Испытательное оборудование ·
СПОАИ · Метрологическое оборудование · Фотоника (Оптика) · ГНСС оборудование.

## ГЛАВНЫЕ ПРАВИЛА (non-negotiable)

1. Светлые, белые поверхности на мягком прохладно-сером фоне; функциональный синий
   акцент. Розовый — **только** для рисков / проблем / болей.
2. Заголовки — Montserrat (700/800/900). Текст — Inter (400/500/600).
3. Каждую секцию открывай UPPERCASE-эйброу-меткой (Montserrat 700, трекинг 0.27em).
4. **Карточки и пиллы вместо буллетов.** Списки превращай в визуальные блоки.
5. Конкретные числа: `67 ГГц`, `<1 ч`, `5 000 ₽` — не «широкий диапазон» / «быстро».
6. **Без эмодзи. Нигде.** Без ALL CAPS в тексте, без курсива, без восклицаний.
7. Заголовки **декларируют** и **не заканчиваются точкой** или вопросом.
8. Щедрый воздух (~60% пустого). Декор — по краям, центр композиции чистый.
9. Иконки — тонкие линейные глифы (Lucide), без заливок. Никаких самодельных SVG
   сложнее круга/ромба.

## ОТСТУПЫ, РАДИУСЫ, ФОРМА

- Шкала отступов: **4 / 8 / 16 / 24 / 32 / 48 / 64 / 96 / 128**. Никаких произвольных
  значений. Внешний паддинг всегда больше внутреннего.
- Радиусы: `8px` инпуты/мелкое · `12px` карточки по умолчанию · `18px` крупные панели ·
  `999px` пиллы/теги. **Острых углов нет.**
- Границы — хайрлайн почти прозрачный (`rgba(30,60,110,0.12)`). Синяя граница
  появляется **только** в active/selected.
- Контент-макс: `1200px`.

## ЭФФЕКТЫ

- **Тени** мягкие, низкоконтрастные, большой блюр — поднимают, а не делят:
  `--shadow-md: 0 10px 30px rgba(20,40,80,0.10)`. Жёстких теней нет.
- **Свечение/выделение:** мягкий синий внешний глоу `0 0 50px rgba(75,159,213,0.18)`
  + синяя граница.
- **Пазл-текстура** (тонкая синяя решётка) на малой непрозрачности (~20%), привязана
  к одному краю и маскируется к центру.
- **Радиальные синие глоу** в углах / за крупными числами — едва заметны, для глубины.
- **Движение:** контент проявляется opacity + лёгкий сдвиг вверх (~600ms),
  `cubic-bezier(0.16,1,0.3,1)`. Декоративные глоу — медленное «дыхание». Без отскоков.
  Уважай `prefers-reduced-motion`.
- **Sticky-хедеры:** полупрозрачный белый фон + `backdrop-filter: blur(12px)`.
- **Hover:** primary-кнопка светлеет (`#1C6BB0 → #3E8FD0`); secondary получает синюю
  границу + синий текст; карточка поднимается на `-4px`. Без сжатия по нажатию.

## КОМПОНЕНТЫ (React)

Восемь примитивов. В среде дизайна подключаются через `_ds_bundle.js`
(`window.DesignSystem_176d4c`). В обычном коде воспроизводи их по этим спецификациям,
используя токены выше.

- **Button** — пилл-кнопка, Montserrat 700.
  `variant: "primary" | "secondary" | "ghost"` · `size: "sm" | "md" | "lg"` ·
  `disabled` · `icon` (тонкий глиф) · `onClick`.
- **Card** — градиентная панель + хайрлайн-граница, базовый контейнер.
  `variant: "default" | "active" | "alert"` (`active` = синяя граница + глоу;
  `alert` = розовая палитра, только риски) · `padding`.
- **Tag** — пилл-чип для категорий/фильтров. `active` (синяя заливка + граница).
- **Badge** — маленький UPPERCASE-маркер статуса.
  `tone: "blue" | "neutral" | "alert"` (`alert` = розовый, только проблемы).
- **Callout** — полупрозрачная синяя полоса: иконка + одна строка акцента. `icon`.
- **Stat** — крупный светло-синий числовой акцент над подписью.
  `value` (конкретное: «67 ГГц», «<1 ч») · `label` · `sublabel` · `align`.
- **Input** — тёмное поле; хайрлайн-граница, синеет в фокусе. `label` · `hint`.
- **Icon** — тонкий линейный глиф из Lucide. `name` (напр. `radio`, `satellite`,
  `activity`, `zap`, `ruler`, `scan-eye`, `shield-check`) · `size` · `stroke`
  (по умолч. светло-синий) · `strokeWidth` (~1.6).

**Иконки:** Lucide через `<script src="https://unpkg.com/lucide@latest"></script>`.
Только тонкие линейные глифы, синий штрих `#62B4EC`. Без эмодзи и юникод-глифов.

## ЛОГОТИП И МОТИВЫ

- Знак — тёмно-синий (`#004180`) круг из четырёх пазлов, разделённых белыми линиями.
- Локап = знак + словознак Montserrat:
  `<span class="akm-logo"><img src=".../logo-mark.png"><span>АКМЕТРОН</span></span>`.
- Пазл-текстура — фирменный мотив, по краю, тает к центру.

---

## КАК АКМЕТРОН ПИШЕТ (тон)

- Уверенно, без напора. Утверждения, а не питч. Деловой регистр с воздухом
  («в один клик», «без посредника»), но без сленга и хайпа.
- Обращение к читателю напрямую («Опишите задачу», «Получите расчёт»); о компании —
  «мы» подразумеваемо.
- **Заголовки декларируют** — без точки и без вопроса в конце.
  Пример: «Измеряем точно. Разрабатываем под задачу».
- **Числа конкретны:** `<1 ч`, `5 000 ₽`, `67 ГГц`; сроки — диапазоны («до 6 месяцев»).
- Каждый термин понятен или объясняется при первом упоминании (СВЧ, ГНСС, СПОАИ).
- Пунктуация: средняя точка `·` разделяет короткие пункты в строке; `—` для вставок;
  избегай точки с запятой.
- Акцент — весом и структурой, не ALL CAPS / курсивом / восклицаниями. Uppercase —
  только для коротких эйброу-меток и бейджей.
- Одна идея на сцену. Строгий порядок: метка → заголовок → подзаголовок → текст.
